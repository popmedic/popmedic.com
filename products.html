Mp4Autotag

Mp4AutoTag is a meta data editor for mp4 file types. In-other-words, it edits the info you see about the mp4 file in iTunes and Finder. Mp4AutoTag atempts to locate your mp4 file by file name in thetvdb.com or themoviedb.org. It then allows you to save the information to the file. You can also edit the results, attach different images, rename the file automatically to the proper standard, and/or add a watermark to the image. Images can be pulled from not only thetvdb.com or themoviedb.org, but also iTunes Store.
