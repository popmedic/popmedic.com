Mp4Split

I have been really busy with a new version of PopMSplit. PopMSplit has a more specified function, which lead to a name change. PopMSplit is now only supported for splitting into mp4 files. If the source file is another format, it will be split and exported into an mp4. This "new more specified function," along with Mp4Autotag's success, made me change the name to "Mp4Split." So, from here on out, PopMSplit will for ever be known as Mp4Split. I completely revamped the GUI, making it look and feel similar to Mp4Autotag. Controlling playback of the editor is now easier, faster, more accurate, and I have big plans for the future.

Lined up for the future of Mp4Split will be:
  • Jump to... control feature that will allow for string command jumping around in the movie.
  • Chapter indexes will be loaded and available to jump to from the menu bar.
  • Segment exclusion from segments table.

PopMedia3.0

Popmedia is an idea and project that I use at my house and want to provide to others. The concept is simple: A transparent media server that uses apache with mongrel to expose directories to various different clients. Using meta-data and the file-system alone (NO INDEXING, SCANNING, DATABASING, nothing, just the files and file system alone). This make a much more friend experience because... you don't have to wait a year to see changes, or spend hours cleaning up bad scans. Currently there is a working HTML client, with zip downloads and access to the servers administration, and a Roku client, with searching, BIF files, and meta-info viewing.

Lined up for the future of PopMedia3.0 will be:
  • Some form of instructions on how to install and use this beast.

Scripts

Added all of my sweet command-line ruby scripts to github. This organized me, and also provided the code to others for use as they see fit. The added scripts are:
  • dvd2mp4 - This script uses vobcopy and ffmpeg to rip a DVD. It is a great example of reading cl stdio and creating your own stdio out of it. It also shows the steps for ripping a DVD on the command-line.
  • m4a2mp3 - This script uses id3lib-ruby and mp4info along with faad and lame to convert a m4a (stupid iTunes audio file, just stick with the damn standard fools'), into a mp3 file. A great example of using id3lib-ruby and the cl application mp4info and mp4art.
  • ffmpeg wrappers - A set of Ruby scripts for dealing with ffmpeg output on the command line. A great example of using popen to run shell executables with captured stdio and personal stdio.
  • Roku Buildbifs - A Ruby script for creating Roku BIF files. A great example of how to make a BIF file with ruby, little endians guys!
